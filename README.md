# 专属男友服务菜单 (Boyfriend Service Menu)

一个充满爱意的微信小程序，提供专属投喂、陪玩求助、紧急联系等功能，用数字化工具提升情侣互动体验。

## ✨ 项目简介
这是一个从零到一独立开发的微信小程序，从需求分析、原型设计到前端开发、真机调试全程独立完成。项目旨在通过轻量化的工具，让情侣间的日常互动更有趣、更高效，同时也作为个人全栈开发与产品思维的实践项目。

## 🎯 已实现功能
- 🏠 **首页轮播**：展示每日情话与服务动态，支持自动播放和手动切换
- 🎮 **专属服务**：奶茶补给、水果拼盘、陪玩服务、紧急求助等分类功能卡片
- 📞 **一键拨号**：快速联系专属男友，紧急情况一键直达
- 📱 **多端适配**：采用Flex弹性布局，完美兼容iOS与Android主流机型

## 🛠️ 技术栈
- **前端框架**：微信小程序原生开发 (WXML / WXSS / JavaScript)
- **布局方案**：Flex 弹性布局，响应式设计
- **架构设计**：组件化开发，模块化代码结构
- **后续规划**：微信云开发 (CloudBase)、数据统计可视化、A/B测试优化

## 📊 数据分析师的设计思路
作为数据科学与大数据技术专业的学生，我在项目中融入了数据驱动的产品思维：
1. **数据采集层**：设计了服务记录与用户反馈模块，为后续分析积累原始数据
2. **数据分析层**：计划接入云数据库，实现投喂次数统计、响应时长分析、满意度趋势等指标
3. **产品迭代层**：将通过A/B测试优化界面布局与交互流程，用数据量化用户体验提升

## 🚀 如何运行
1.  下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2.  克隆本项目到本地
3.  在微信开发者工具中导入 `专属小程序` 文件夹，填入你的小程序AppID
4.  点击"编译"即可在模拟器中预览效果
